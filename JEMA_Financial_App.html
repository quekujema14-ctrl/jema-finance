🔗 Connect to Google Sheets

Follow these steps once to link this app to your Google Sheet (quekujema14@gmail.com). Changes in either place will sync automatically.

  • 1
    Open Google Cloud Console
    Go to console.cloud.google.com while signed in as quekujema14@gmail.com. Create a new project (e.g. "JEMA Financial App").
  • 2
    Enable Google Sheets API
    In the left menu go to APIs & Services → Library. Search for Google Sheets API and click Enable.
  • 3
    Configure OAuth Consent Screen
    Go to APIs & Services → OAuth consent screen. Choose External, fill in app name (e.g. "JEMA App"), your email, and save. Add your Google account as a test user.
  • 4
    Create OAuth Client ID
    Go to APIs & Services → Credentials → Create Credentials → OAuth Client ID. Choose Web Application. Under Authorized JavaScript Origins add:
    https://finance.jemastudios.com http://localhost:8080
    💡 Live app URL: https://finance.jemastudios.com
